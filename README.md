## 📦 Mi proyecto: [DocTriX](https://github.com/MiguelMillones/proyecto_PDF)

**Una herramienta profesional para gestionar documentos PDF**

Este proyecto nació de una necesidad real:
> *"¿Por qué no hay una herramienta gratis, sin anuncios y que funcione offline para gestionar PDFs?"*

**Así que me puse a construirla yo mismo.**

### El proceso (como buen ingeniero)

1. **Definí el problema:** Necesitaba extraer, unir y foliar páginas PDF
2. **Investigué soluciones:** PyPDF2, ReportLab, PyQt6
3. **Prototipé rápido:** Un script de consola primero
4. **Iteré hasta mejorar:** Añadí interfaz, barra de progreso, estilos
5. **Empaqueté el producto:** Ejecutable .exe de 81 MB

**¿El resultado?** Una app que uso a diario y que funciona mejor que muchas opciones comerciales.
